# Sweet Chili

This recipe makes a chili.

### Bakeware

* Sheet pan
* Skillet
* Slow cooker

### Baking

#### Bacon
400ºF for 25 minutes

#### Chili
8 hours on low in the slow cooker

### Ingredients

| Quantity | Ingredient |
| -------- | ---------- |
| 2 lb | ground beef |
| 1 | onion |
| 2 sticks | celery |
| 1 | green pepper |
| 15oz | can of diced tomatoes |
| 15oz | can of tomato sauce |
| 1 pkg | bacon |
| 2 cans | beans |
| ⅓ cup | ketchup |
| 1 tbsp | garlic
| 1 tbsp | chili powder |
| 2 tsp | paprika |
| 1 tbsp | brown sugar |
| 1 tsp | cumin |
| 1 tsp | oregano |
| 1 tsp | cayenne pepper |

### Instructions

* Cook bacon on rack in oven
  * Chop bacon into bits
  * Add to slow cooker
* Brown beef
  * Add to slow cooker
